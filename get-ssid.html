APRS SSID Reference

Common APRS SSID identifiers and the typical roles they represent.

SSID Description
-0Primary station (fixed and message capable)
-1Digipeater or home station running a relay
-2Second digipeater (on 70CM)
-3Third digipeater
-4HF to VHF gateway
-5IGate (not a home station)
-6Operations via satellite
-7Kenwood D7 handheld
-8Boats, sailboats, or secondary mobile units
-9Primary mobile station
-10Internet-only operation
-11APRS touch-tone users (balloons, etc.)
-12Portable units or one-way trackers
-13Reserved for future use
-14Reserved for future use
-15Generic additional station, digipeater, etc.

These are conventional APRS SSID uses, not hard technical limits.
